# Install Clustta

Clustta has two main interfaces:

- **The desktop app** — Where you actually do your work. Available for Windows, macOS, and Linux.
- **The web dashboard** — At [app.clustta.com](https://app.clustta.com). Used for managing your account, studios and collaborators.

You can sign up for an account from either.

## Download the desktop app

Pick the install method that fits your platform:

### Windows

[**Microsoft Store**](https://apps.microsoft.com/detail/9PNRGHGP3LGX) — recommended. Auto-updates, sandboxed install, no admin rights needed.

Or grab the standalone installer from [clustta.com/download](https://clustta.com/download).

### macOS

[**Mac App Store**](https://apps.apple.com/us/app/clustta/id6748349288) — recommended for both Intel and Apple Silicon Macs.

Or grab the `.dmg` from [clustta.com/download](https://clustta.com/download).

### Linux

Download the `.deb` (Debian/Ubuntu) or `.AppImage` from [clustta.com/download](https://clustta.com/download).

For Debian/Ubuntu:

```bash
sudo dpkg -i clustta_*.deb
```

For the AppImage:

```bash
chmod +x Clustta-*.AppImage
./Clustta-*.AppImage
```

## Create an account

You'll need a Clustta account to sync projects, collaborate, or use a Clustta Cloud studio. Personal/offline use also requires sign-in but doesn't push anything to the cloud.

1. Open the desktop app (or visit [app.clustta.com](https://app.clustta.com)).
2. Click **Sign up** and provide your name, email and password.
3. Verify your email when the confirmation arrives.
4. Sign in.

After signing in for the first time, you'll land in your **Personal studio** — your private, offline-only workspace. Every Clustta user has one.

::: tip
Personal studio projects never leave your machine. They're perfect for solo work, experiments, or trying Clustta out before bringing your team on.
:::

## Switching studios

Once you have access to more than one studio (your Personal studio + any team studios you create or get added to), use the dropdown at the top-left of the app — labelled with the current studio name — to switch.
