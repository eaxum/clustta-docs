# Keyboard Shortcuts

Quick access to the operations you'll use most. Shortcuts work in the desktop client when the relevant view has focus.

## File & Project Management

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New Project |
| `Ctrl+S` | Sync Project |

## Content Creation

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | New Collection |
| `Ctrl+T` | New Task (asset) |

## Item Operations

| Shortcut | Action |
|----------|--------|
| `F2` | Rename selected item |
| `Ctrl+F2` | Edit selected item metadata |
| `↵ Return` | Open / enter selected item |
| `Delete` | Free up local file (keeps history) |
| `Shift+Delete` | Soft-delete (send to trash) |

## Versioning

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+C` | Create checkpoint on selected asset |

## Navigation & Search

| Shortcut | Action |
|----------|--------|
| `F3` | Open search |
| `F5` | Reload current view |

::: tip
Most operations also have right-click menu equivalents and toolbar buttons. Shortcuts are an accelerator, not the only way in.
:::
