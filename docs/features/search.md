# Search

Find anything in your project - assets, collections, tags - by name, in milliseconds.

## How to search

Press **`F3`** or click the search box at the top of the browser. Start typing.

Results update as you type and include:

- **Assets** matching by name, type, or tag
- **Collections** matching by name or type
- **Recent matches** at the top

Click any result to jump straight to it in the browser.

## What's searched

- Asset names
- Collection names
- Tag names
- Asset and collection types
- Optional: comments and descriptions (depending on view filter)

Search is **scoped to the current project** by default. You can broaden or narrow scope from the search filters.

## Filters

The toolbar above the browser lets you stack filters in addition to text search:

- **Type** filter - restrict to assets of a certain Asset Type (Animation, Texture, etc.)
- **Tag** filter - restrict to assets with specific tags
- **Status** filter - restrict by status (Todo / WIP / WFA / Retake / Done)
- **Assignee** filter - restrict to one user's tasks
- **Extension** filter - restrict to specific file types (`.blend`, `.psd`, etc.)

Filters combine - applying type + status + assignee gives you "all Animation tasks in WFA assigned to Adaeze" instantly.

## Saved views (My Tasks, Assigned, custom)

The browser has built-in tabs:

- **Default** - everything in the current scope
- **My Tasks** - tasks assigned to you across the whole project
- **Assigned tasks** - every assigned task (filter further by user)

You can open additional filtered tabs (e.g. "All Characters") and they persist for the session.

## Tips

- **Use type prefixes mentally.** "Anim..." filters quickly to Animation tasks; "char..." to Characters collections.
- **Combine search + filters.** Type a name, then filter by status to slice further.
- **Search across the project, not just the current collection.** Search is project-wide by default - useful when you forget where something lives.
